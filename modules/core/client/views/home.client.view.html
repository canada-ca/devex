Government of Canada Developers Exchange

About GC DevEx

The Government of Canada Developers Exchange (GC DevEx) is a website designed to make it easier to collaborate on open source digital solutions. Teams within the Government of Canada can post contract opportunities, and developers can find contract opportunities. The dollar value of these contracts cannot exceed $9,999 CAD. All procured code is licensed as open source and housed on GitHub to encourage re-use.

Register now to be notified of future opportunities or follow the GC DevEx twitter bot! For more information, see the GC DevEx User Guide.

Participating government departments

GC DevEx was launched by Privy Council Office and Treasury Board of Canada Secretariat, in collaboration with Public Services and Procurement Canada.

If you wish to post opportunities, sign up with your GitHub account and follow the instructions in the GC DevEx User Guide.

Note: this is an experiment whose future is uncertain. If you think that GC DevEx can meet a need that you have, feel free to make it work for you. For example, post a project idea on GitHub.

Re-use this code

The code for the GC Dev Ex platform was first developed by the Government of British Columbia for its British Columbia Developers' Exchange. It was then modified to meet Treasury Board of Canada Secretariat (TBS) Web Standards and the Official Languages Act.

Other governments can re-use this modified code. With the exception of the federal identifier (AKA Canada logo), all parts of the platform are licensed for re-use through the Apache License.

Whether you are part of a provincial, municipal, or national government elsewhere, you can fork the code on GitHub and set-up your own website to work with developers in your community.
